API リファレンス

REST APIで口コミ取得を自動化。Base URL: https://review-scraper-199337986055.asia-northeast1.run.app

スクレイピング開始

POST/scrape
スクレイピングジョブを非同期で開始。即座にjob_idが返ります。
リクエスト
curl -X POST /scrape \ -H "Content-Type: application/json" \ -d '{"url": "https://www.google.com/maps/place/...", "source": "gmap"}'
レスポンス 202
{"job_id": "a1b2c3d4", "status": "running"}

ジョブ状態確認

GET/jobs/{job_id}
進捗・結果を取得。done 時に reviews 配列を含む。
レスポンス（実行中）
{"status": "running", "progress": 45, "message": "スクロール中... 45件取得"}
レスポンス（完了）
{"status": "done", "review_count": 150, "reviews": [{"review_id":"x","author":"田中","rating":"5","date":"1か月前","comment":"最高！"}, ...]}

その他

GET/jobs/{job_id}/csv
完了ジョブをCSV形式でダウンロード
GET/jobs/{job_id}/logs
タイムスタンプ付き実行ログ取得
GET/jobs
全ジョブ一覧
DELETE/jobs/{job_id}
ジョブとデータを削除

Python サンプル

import requests, time BASE = "https://review-scraper-199337986055.asia-northeast1.run.app" # 1. Start resp = requests.post(f"{BASE}/scrape", json={"url": "https://...", "source": "gmap"}) job_id = resp.json()["job_id"] # 2. Poll while True: s = requests.get(f"{BASE}/jobs/{job_id}").json() print(f"{s['message']} ({s['review_count']}件)") if s["status"] in ("done", "failed"): break time.sleep(3) # 3. Result if s["status"] == "done": print(f"完了: {len(s['reviews'])}件")
