API リファレンス

REST APIで口コミ取得を自動化できます。Base URL: https://review-scraper-199337986055.asia-northeast1.run.app

スクレイピング開始

POST /scrape
スクレイピングジョブを非同期で開始します。即座にjob_idが返ります。
リクエスト
curl -X POST https://review-scraper-199337986055.asia-northeast1.run.app/scrape \ -H "Content-Type: application/json" \ -d '{"url": "https://www.google.com/maps/place/...", "source": "gmap"}'
レスポンス (202 Accepted)
{"job_id": "a1b2c3d4", "status": "running"}

ジョブ状態確認

GET /jobs/{job_id}
ジョブの進捗・結果を取得。status が "done" のとき reviews 配列が含まれます。
リクエスト
curl https://review-scraper-199337986055.asia-northeast1.run.app/jobs/a1b2c3d4
レスポンス (実行中)
{"job_id": "a1b2c3d4", "status": "running", "progress": 45, "message": "スクロール中... 45件取得", "review_count": 45}
レスポンス (完了)
{"job_id": "a1b2c3d4", "status": "done", "review_count": 150, "reviews": [ {"review_id": "xxx", "author": "田中太郎", "rating": "5", "date": "1か月前", "comment": "最高でした！"}, ... ]}

CSV ダウンロード

GET /jobs/{job_id}/csv
完了済みジョブの結果をCSV形式でダウンロード。
リクエスト
curl -o reviews.csv https://review-scraper-199337986055.asia-northeast1.run.app/jobs/a1b2c3d4/csv

実行ログ

GET /jobs/{job_id}/logs
ジョブの実行ログ（タイムスタンプ付き）を取得。
レスポンス
[ {"time": "2026-03-06T10:15:29Z", "message": "セッション開始中... (試行 1/5)"}, {"time": "2026-03-06T10:15:35Z", "message": "Cookie取得中..."}, {"time": "2026-03-06T10:16:10Z", "message": "スクロール中... 15件取得"}, ... ]

ジョブ一覧

GET /jobs
全ジョブの一覧（レビューデータなし）。新しい順にソート。
リクエスト
curl https://review-scraper-199337986055.asia-northeast1.run.app/jobs

ジョブ削除

DELETE /jobs/{job_id}
ジョブとレビューデータを削除。
リクエスト
curl -X DELETE https://review-scraper-199337986055.asia-northeast1.run.app/jobs/a1b2c3d4

Python サンプル

import requests, time BASE = "https://review-scraper-199337986055.asia-northeast1.run.app" # 1. Start scraping resp = requests.post(f"{BASE}/scrape", json={ "url": "https://www.google.com/maps/place/...", "source": "gmap" }) job_id = resp.json()["job_id"] # 2. Poll for completion while True: status = requests.get(f"{BASE}/jobs/{job_id}").json() print(f"{status['message']} ({status['review_count']}件)") if status["status"] in ("done", "failed"): break time.sleep(3) # 3. Download results if status["status"] == "done": reviews = status["reviews"] print(f"取得完了: {len(reviews)}件")
