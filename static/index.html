スクレイピング開始

POST/scrape
スクレイピングジョブを非同期で開始。即座に job_id が返ります。
リクエスト
curl -X POST https://review-scraper-199337986055.asia-northeast1.run.app/scrape \ -H "Content-Type: application/json" \ -d '{"url": "https://www.google.com/maps/place/...", "source": "google"}' # source: "google" (Google Maps) / "tripadvisor" (TripAdvisor)
リクエストボディ
{ "url": string, // 必須。Google Maps または TripAdvisor の URL "source": string // 必須。"google" | "tripadvisor" }
レスポンス 202 Accepted
{ "job_id": "a1b2c3d4", // string: ジョブ固有ID（8文字hex） "status": "running" // string: 初期ステータス }
エラーレスポンス
400: {"detail": "url is required"} // URLまたはsource未指定 400: {"detail": "Unsupported URL domain"} // 対応外ドメイン 409: {"error": "同一URLが実行中です", ...} // 5分以内に同じURL実行中

ジョブ状態確認

GET/jobs/{job_id}
ジョブのメタデータと進捗を取得。レビュー本体は /jobs/{job_id}/reviews で取得。
レスポンス（実行中）
{ "job_id": "a1b2c3d4", "status": "running", "progress": 45, "message": "スクロール中... 45件取得", "review_count": 45 }
レスポンス（完了）
{ "job_id": "a1b2c3d4", "status": "done", "review_count": 150, "duration": 243, "message": "完了: 150件取得" }
レスポンス（失敗）
{ "job_id": "a1b2c3d4", "status": "failed", "review_count": 0, "duration": 386, "message": "エラー: ...", "error": "Google Maps レビュー取得失敗 (5回リトライ済み)" }
フィールド
job_id : string — ジョブ固有ID status : string — "running" | "done" | "failed" progress : number — 現在の取得件数 review_count : number — 最終取得件数 message : string — 現在のステップ説明 duration : number? — 所要秒数（完了/失敗時のみ） error : string? — エラー詳細（失敗時のみ） url : string — 対象URL source : string — "google" | "tripadvisor" created_at : string — ジョブ作成日時（ISO 8601）
エラー
404: {"error": "Job not found"}

実行ログ

GET/jobs/{job_id}/logs
タイムスタンプ付きの実行ログ。各ステップ（Cookie取得、CAPTCHA検出、リトライ等）を記録。
レスポンス
[ {"time": "2026-03-06T13:00:00Z", "message": "セッション開始中... (試行 1/5)"}, {"time": "2026-03-06T13:00:05Z", "message": "Cookie取得中..."}, {"time": "2026-03-06T13:00:15Z", "message": "スクロール中... 45件取得"} ]
フィールド
time : string — タイムスタンプ（ISO 8601） message : string — ログメッセージ

全ジョブ一覧

GET/jobs
全ジョブの一覧を返します（最新順）。各要素は GET /jobs/{job_id} と同じ形式。
レスポンス 200
[ { "job_id": "a1b2c3d4", "url": "https://www.google.com/maps/place/...", "source": "google", "status": "done", "review_count": 150, "duration": 243, "message": "完了: 150件取得", "created_at": "2026-03-06T15:35:30.541131+00:00", "error": null }, ... ]

レビュー取得

GET/jobs/{job_id}/reviews
完了ジョブの口コミ一覧をJSON配列で返します。
レスポンス
[ {"review_id": "abc123", "author": "田中太郎", "rating": "5", "posted_at": "2026-02-04", "comment": "最高のお店！"}, {"review_id": "def456", "author": "鈴木花子", "rating": "4", "posted_at": "2026-01-15", "comment": "雰囲気が良い"}, {"review_id": "ghi789", "author": "佐藤次郎", "rating": "3", "posted_at": "2025-12-01", "comment": ""} ]
フィールド
review_id : string — レビュー固有ID（サイト側のID） author : string — 投稿者名 rating : string — 評価（"1"〜"5"） posted_at : string — 投稿日（ISO 8601: YYYY-MM-DD） comment : string — コメント本文（星のみの場合は空文字 ""）
エラー
404: {"error": "Job not found"} 400: {"error": "Job not finished"} // status が done 以外

ジョブ削除

DELETE/jobs/{job_id}
ジョブとそのレビューデータ・ログを完全に削除します。
レスポンス 200
{"ok": true}

ステータス一覧

running : スクレイピング実行中（ポーリングで進捗確認可能） done : 正常完了（レビュー取得可能） failed : 失敗（error フィールドに原因）
失敗の主な原因
• IP制限（クチコミタブが表示されない） • CAPTCHA検出（TripAdvisor DataDome） • 10分タイムアウト（全体の実行時間超過） • ページ読み込み失敗（ネットワークエラー） • 5回リトライ後も取得不可

Python サンプル

import requests, time BASE = "https://review-scraper-199337986055.asia-northeast1.run.app" # 1. スクレイピング開始 resp = requests.post(f"{BASE}/scrape", json={ "url": "https://www.google.com/maps/place/...", "source": "google" # or "tripadvisor" }) job_id = resp.json()["job_id"] print(f"Job started: {job_id}") # 2. ポーリングで進捗確認 while True: s = requests.get(f"{BASE}/jobs/{job_id}").json() print(f"[{s['status']}] {s['message']} ({s.get('review_count',0)}件)") if s["status"] in ("done", "failed"): break time.sleep(3) # 3. 結果取得 if s["status"] == "done": print(f"完了: {s['review_count']}件 ({s.get('duration',0)}秒)") # JSON download reviews = requests.get(f"{BASE}/jobs/{job_id}/reviews").json() print(f"{len(reviews)} reviews")

ソース指定

google — Google Maps の口コミ
tripadvisor — TripAdvisor の口コミ

URLを元に自動判定もされます。明示的に指定する場合は source パラメータを使用してください。
